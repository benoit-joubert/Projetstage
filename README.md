# Application des recommandés

Nous sommes un étudiant de DUT et un étudiant de la MIAGE réalisant une application permettant de gérer la diffusion des recommandés du service de l'urbanisme. Elle permettra de faciliter la vie des personnes de ce service et d'utiliser des technologies plus récentes que l'application actuelle qui est sous Access97.

## Languages Used

* HTML 5
* CSS 3
* PHP 7.3
* Javascript
* JQuery

## Built With

* [PHPStorm](https://www.jetbrains.com/phpstorm/) - The IDE used
* [Apache](https://httpd.apache.org/) - The Open-source HTTP server
* [Alwaysdata](https://www.alwaysdata.com/fr/) - The temporary host of the forum
